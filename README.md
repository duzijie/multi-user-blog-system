# 多用户易览博客系统

## 欢迎使用我们的博客系统！


本系统是一个支持多用户注册的博客系统，具有写博客、浏览其他人博客、留言、点赞等功能。由Vue.js 2.0、Spring boot、Spring Security JWT、 Mybatis Plus等技术构成。
可访问 http://112.74.34.248/ 预览博客系统的一部分内容（管理系统暂无预览版，可查看文档下方图片进行简要浏览）。
